# Mercury Project Context

## Project Overview

Mercury is a personal golf intelligence platform.

The goal is to build a golf decision-support system that combines player tendencies, equipment data, course information, weather, lie conditions, historical performance, and golf strategy into personalized recommendations and analytics.

Mercury is intended to replace the functionality of apps like 18Birdies while providing significantly deeper personalization and analysis.

Mercury should help the golfer make better decisions without becoming dependent on the system.

Guiding principle:

"Mercury informs decisions. Mercury does not make decisions."

## Current UI Direction

Desired characteristics:

* Professional
* Minimal
* Analytical
* Premium
* Fast

Avoid:

* Cartoonish styling
* Mobile game aesthetics
* Excessive colors
* Overly playful design

Design inspiration:

* Apple
* Arccos
* PGA Tour
* Professional analytics platforms

The application should feel like a premium golf intelligence platform rather than a casual golf app.

---

## Long-Term Vision

Mercury Golf is the first implementation of a broader Mercury platform.

Potential future projects include:

* Fantasy football intelligence
* Professional golf analytics
* Personal finance analytics
* Geography and migration analytics
* Other decision-support systems

The long-term objective is to build a reusable framework for decision intelligence applications.
