# Equipment Profile

Mercury should track equipment historically so performance can be measured instead of guessed.

The current live equipment setup is stored in [Player Profile](player-profile.md). This file describes how Mercury should evaluate equipment over time.

## Equipment Categories

- Driver
- Mini driver
- Fairway woods - might not have any soon
- Long irons
- Mid irons
- Short irons
- Wedges
- Putter
- Golf ball

## Current Source Of Truth

Current player equipment details, shafts, grips, golf ball, and carry windows should live in [Player Profile](player-profile.md).

This avoids splitting the same facts across multiple files.

## Equipment Questions

- Is mini driver outperforming driver?
- Which driver produces better scoring outcomes?
- Which ball performs best?
- Is 4 iron or 7 wood more useful?
- Which equipment setup creates fewer penalties?
- Which setup produces better scoring, not just better feel?

## Metrics To Track

- Rounds used
- Shots hit
- Fairway outcomes
- Green outcomes
- Penalty outcomes
- Miss direction
- Typical distance
- Confidence
- Score impact

## Important Principle

Equipment should be judged by scoring impact and decision value, not only by the best shots it can produce.
