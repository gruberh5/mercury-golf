# Game Profile Input Guide

This is an optional guide for what to add to [Player Profile](player-profile.md).

The actual source of truth is [Player Profile](player-profile.md).

## How To Use This File

Use this file as a checklist. Write the real notes in [Player Profile](player-profile.md).

Write naturally. It does not need to sound technical.

Good entries can look like:

> My driver usually starts right. If it overdraws, it can turn into a hook. I almost never pull it left from the start.

## Bag Setup

List every club currently in the bag.

For each club, include:

- Brand and model if known
- Loft if known
- Typical carry distance
- Typical total distance
- Full swing or partial swing use
- Confidence level
- Common miss

## Golf Ball

Describe the ball being used.

Include:

- Ball model
- Why you play it
- Any known tendencies
- Other balls you want to compare

## Driver Profile

Describe:

- Swing speed
- Launch tendency
- Spin tendency
- Start line
- Shot shape
- Common misses
- Confidence level
- When you choose driver
- When you avoid driver

## Mini Driver Profile

Describe:

- Why you trust it
- Typical use cases
- Distance compared to driver
- Miss pattern
- Confidence level

## Iron Profile

Describe:

- Delivery style
- Contact pattern
- Common miss
- Long iron confidence
- Rough performance
- Distance control

## Wedge Profile

Describe:

- Full wedge comfort
- Partial wedge comfort
- Chipping comfort
- Sand comfort
- Common miss

## Putting Profile

Describe:

- Putter model
- Confidence level
- Strengths
- Weaknesses
- Distance control
- Short putting
- Lag putting

## Course Management Tendencies

Describe:

- When you get too aggressive
- When you play too conservatively
- Pins you like to attack
- Pins you should avoid
- Tee shots that make you uncomfortable
- Yardages that feel easy
- Yardages that feel awkward
